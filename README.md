# Практическая работа №2: Веб-скрапер для khpet27.ru

Программа для парсинга 100 объектов с сайта https://khpet27.ru и сохранения их в базу данных SQLite.

## Структура данных

Каждый объект содержит следующие поля:
- **ID** - Уникальный идентификатор (1-100)
- **Name** - Название статьи/объекта
- **Audio** - URL аудиофайла (сгенерирован)
- **Image** - URL изображения
- **Text** - Текстовое описание

## Установка и запуск

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Запустите скрапер:
```bash
python khpet27_scraper.py
```

3. Проверьте данные:
```bash
python verify_data.py
```

## Функциональность

- Автоматическое создание базы данных SQLite (`khpet27_data.db`)
- Парсинг реальных статей с сайта khpet27.ru
- Генерация дополнительных данных при необходимости
- Сохранение 100 объектов в таблицу `objects`
- Отображение первых 10 записей
- Обработка ошибок и задержки между запросами

## База данных

Таблица `objects`:
```sql
CREATE TABLE objects (
    id INTEGER PRIMARY KEY,
    name TEXT NOT NULL,
    audio TEXT,
    image TEXT,
    text TEXT
);
```
